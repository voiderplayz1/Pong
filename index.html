🔥 Firebase Setup — PaintChat

Paste your Firebase config object from the Firebase Console below.
It looks like a JavaScript object starting with {.

📋 How to find it:
Firebase Console → Your Project → ⚙️ Project Settings → Your apps → SDK config → select Config → copy the object
🔒 Firestore rules to set:
In Firestore → Rules, paste:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /rooms/{r}/{doc=**} { allow read, write: if true; } } }
